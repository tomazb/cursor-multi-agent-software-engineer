# v0.2 Local Hardening Implementation Plan

> **For agentic workers:** Execute task-by-task with TDD. Steps use checkbox syntax.

**Goal:** Implement ROADMAP v0.2 (GitHub issue #2): durable store, git isolation, markers, redaction, budgets, recovery, schemas, and 0.2.0 packaging.

**Architecture:** Extract `RunStore`; harden `FileRunStore`; add git workspace manager; gate roles on markers; redact secrets; prefer stdin prompts; add retry/supersede; ship JSON schemas + lockfile CI.

**Tech Stack:** TypeScript ESM, Node test runner, no new runtime dependencies.

**Spec:** `docs/superpowers/specs/2026-07-22-v0.2-local-hardening-design.md`

## Global Constraints

- Keep transitions in `src/state-machine.ts`
- Keep provider code in `src/runtimes/`
- Untrusted text never becomes shell commands
- Every behavior change needs tests + docs
- Run `npm run check` before claiming done
